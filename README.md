# LiveSplit.ImOut
> "You know what, I can sum up where I stand on this already. It's pretty disturbing, and it freaks me out. I'm already out." 

\- Daymond John


A LiveSplit plugin that displays "I'm Out!" or "I'm In!" depending on the pace of your run.
![Preview Image](https://raw.githubusercontent.com/Hyperfied/LiveSplit.ImOut/main/LiveSplit.ImOut/Components/preview.png)

## Setup
First, download the [component .dll file](https://raw.githubusercontent.com/Hyperfied/LiveSplit.ImOut/main/LiveSplit.ImOut/Components/LiveSplit.ImOut.dll) and place it inside LiveSplit's component folder. Then download the [Resources.zip](https://raw.githubusercontent.com/Hyperfied/LiveSplit.ImOut/main/LiveSplit.ImOut/Components/Resources.zip) and put the contents inside the LiveSplit resources folder.

You also need to obtain the Gotham Black font and also place that inside of the Resources folder. Ensure the font is named "Gotham Black.otf" inside of the folder.

Once done, edit your LiveSplit layout and add the I'm Out! component from the Media tab.

Happy running!
